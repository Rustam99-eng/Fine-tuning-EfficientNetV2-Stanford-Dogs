# Fine-tuning-EfficientNetV2-Stanford-Dogs
Модели EfficientNetV2 представляют собой следующее поколение свёрточных архитектур для задач компьютерного зрения, созданное компанией Google на базе предыдущей линейки EfficientNet. Их ключевые особенности — применение гибридных свёрточных блоков (Fused-MBConv) и метода прогрессивного обучения, что заметно ускоряет процесс тренировки и повышает точность при меньших вычислительных затратах. Такие сети часто служат основой для решения задач классификации, обнаружения объектов и семантической сегментации изображений.
В данном решении показан процесс адаптации предобученной модели EfficientNetV2 под конкретную задачу с использованием датасета [Stanford Dogs](https://storage.yandexcloud.net/academy.ai/stanford_dogs.zip).
С целью повышения качества модели были задействованы три функции обратного вызова (callbacks): EarlyStopping, ModelCheckpoint и ReduceLROnPlateau. Кроме того, использование методов аугментации данных позволило добиться высоких показателей эффективности.
Финальная точность после дообучения:
- Обучающая выборка: 0.76
- Валидационная выборка: 0.86
- Оценка на тестовой выборке: 0.86
